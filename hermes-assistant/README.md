# hermes-assistant

**Maps to:** Hermes Prep itinerary (Track 2, all days) — see `../GOALS.md` for full context.

## Goal

Learn how to use Hermes Agent (Nous Research's open-source, self-improving AI
agent), understand why its architecture is powerful, and practice it on real
work-adjacent tasks — email/calendar triage and meeting summaries — before
starting at Link Ventures.
